# Moka-Fait-Trop-De-Niaiseries
Mini-jeu amusant où tu aides Moka le lapin à réparer ses bêtises ! En moins d’une minute, plonge dans une aventure simple, drôle et rapide. Parfait pour rire et s’amuser, ce jeu prolonge l’univers de la bande dessinée. Une fois terminé, découvre la suite des aventures de Moka dans le livre !
